PHP Handler    

MIME Types: application/x-httpd-php

Methods: *

Priority: 10

Suggested Virtual Resources:

  • VRes: pattern="*.php" type="application/x-httpd-php" real=true;
    Any file ending with .php will be processed by the PHP Handler.
Virtual Resource Extras: None

PHP is a server-side, cross-platform, HTML embedded scripting language.

You will have to install PHP separately if you want to use it. The "php" executable must reside in /boot/home/config/bin. The PHP Handler has been tested with PHP 4.0.4pl1 for BeOS (a link to this page can also be found under the "Help" menu in the Robin Hood Console).

All .php files should now work without further wrappers or shebang lines, but they still have to be marked executable. Robin Hood includes a few simple PHP examples (located in the "examples" subfolder of the Robin Hood directory) that can be used to check if everything is set up correctly.

Visit the PHP web site for more information on using and writing PHP scripts, as well as links to sites where you can download useful PHP scripts.
